# Code of Conduct

## Our Pledge

We want this project to be open, respectful, and useful for students, contributors, and maintainers.
Participants are expected to engage professionally and constructively in issues, pull requests, discussions, and reviews.

## Expected Behavior

- Be respectful and patient.
- Assume good intent, but communicate clearly.
- Give actionable technical feedback.
- Keep discussions focused on the project.
- Respect differing levels of experience.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Personal attacks or insulting language.
- Trolling, deliberate disruption, or bad-faith arguments.
- Publishing private information without permission.
- Any conduct that makes participation unsafe or hostile.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, commits, or other contributions that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact a repository maintainer directly through GitHub and request a private conversation. Do not open a public issue for personal safety concerns.

## Scope

This code of conduct applies across the repository and associated project spaces, including issues, pull requests, and discussions.
